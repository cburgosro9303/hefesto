System Monitor &
Network Inspector

A cross-platform desktop application for real-time system monitoring, process management, and network port analysis. Built with Rust and Slint.

Download Source Code

Features

🌐

Network Explorer

Discover all listening ports, filter by TCP/UDP, search by process name or port, and identify exposed services with auto-refresh.

⚙️

Process Explorer

Browse all running processes sorted by CPU, memory, PID, or name. Search, filter, and terminate processes directly.

📈

Process Monitor

Track a single process in real-time with CPU and memory history graphs, I/O metrics, uptime, and state monitoring.

🔧

Developer Tools

Built-in Base64 encoder/decoder with URL-safe support, and JSON formatter, compactor, and validator.

🎨

Themes

Dark and light themes based on the Catppuccin color palette. Switch instantly from preferences.

🌍

Internationalization

Full English and Spanish language support with instant switching across the entire interface.

Built With

Rust Slint 1.9 Tokio sysinfo serde native-tls
